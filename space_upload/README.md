# Parakh - trap-aware candidate ranker

Upload a small candidates.jsonl (up to 100 profiles); the ranker runs fully offline on CPU and returns a ranked CSV with grounded reasons. Built for the Redrob India Runs Data & AI Challenge by Team Evolve.
